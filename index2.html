1.git init

用于创建新仓库，步骤为：创建新目录，进入目录，然后执行 git init以创建新的 git 仓库。 生成 .git 目录，该目录就是本地仓库

2.git add

git add后加文件路径，将文件保存到暂存区

2.git commit -v

在了解git commit -v 之前先了解 git commit -m"", git add 命令只是将文件提交到缓存区，如果要实际提交还需要使用 git commit 命令，记录对存储库的更改，具体为 git commit -m"本次改动的叙述" ，git commit -m "XXX"可以提交修改信息 -v(verbose: 详细的，冗余的)，显示HEAD提交与提交的内容之间的统一差异
